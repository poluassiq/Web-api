Асинхронный backend-сервис для мониторинга погодных данных с использованием FastAPI, WebSocket и NATS.

Основные функции

REST API – полный CRUD для работы с погодными данными
WebSocket – real-time уведомления об изменениях
Фоновые задачи – автоматическое обновление данных каждые 5 минут
NATS интеграция – обмен сообщениями между компонентами
Асинхронная архитектура – высокая производительность
